# วิธีการรันโปรเจค

อัพเกรดเวอชั่น npm ให้เป็นเวอชั่นล่าสุดก่อน ถึงจะรันโปรเจคนี้ได้

`npm update -g npm`

แล้วก็ลง Dependency

`npm install`

คำสั่งรันโปรเจค ต้องเลือก DB_TYPE = "NONE" ไม่งั้นจะรันไม่ได้ เพราะว่าโปรเจคตัวนี้เค้าเชื่อมต่อ MongoDB เอาไว้

`DB_TYPE=“NONE” npm run dev`
